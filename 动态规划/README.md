# 重点突出

1. 动态规划是什么？

   借用胡能尧大佬的话：动态规划是一种思想

   动态规划的两大要素：状态 、 转移

   算法中的每一个状态能否表示出来？状态间的转移方程是否可以给出？能解决这两个问题，那么就可以使用动态规划。
